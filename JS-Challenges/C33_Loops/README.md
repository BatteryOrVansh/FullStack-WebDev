# JavaScript Challenge 33 — Loops

### **Task 1: Sum of First N Natural Numbers**
Write a function `sumOfN(n)` that calculates and returns the sum of all natural numbers from 1 up to `n` using a loop.

### **Task 2: Multiplication Table**
Write a function `printMultiplicationTable(n)` that generates a multiplication table for `n` (from 1 to 10). Each entry in the returned array should be a string in the format: `5 * 1 = 5`.

### **Task 3: Count Vowels in a String**
Write a function `countVowels(str)` that counts the number of vowels (`a, e, i, o, u`) in a given string. The function should be case-insensitive, identifying both lowercase and uppercase vowels.
