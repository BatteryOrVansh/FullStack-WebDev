# JavaScript Challenge 31 — Playing with Variables

### **Task: Mathematical Operations**
Write a function `performMath` that takes two numbers `a` and `b` and returns an object containing the results of the following operations:

* **Add**: The sum of `a` and `b`.
* **Subtract**: The result of `a - b`.
* **Multiply**: The product of `a` and `b`.
* **Divide**: The quotient of `a / b`.
* **Increment**: The value of `a` plus one.
* **Decrement**: The value of `b` minus one.
* **Remainder**: The modulo of `a % b`.
