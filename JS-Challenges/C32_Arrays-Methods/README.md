# JavaScript Challenge 32 — Arrays and Methods

### **Task 1: Array Filtering**
Write a function `filterNumbers(arr)` that returns only numbers from a mixed array (e.g., strings, booleans, and numbers).

### **Task 2: Array Reversal**
Write a function `reverseArray(arr)` that reverses the given array using an inbuilt method.

### **Task 3: Find Maximum in an Array**
Write a function `findMax(arr)` that returns the largest number in the array using a mathematical utility.

### **Task 4: Remove Duplicates from an Array**
Write a function `removeDuplicates(arr)` that returns a new array with all duplicates removed.

### **Task 5: Flatten a Nested Array**
Write a function `flattenArray(arr)` that takes a nested array (multi-dimensional) and returns a single flattened array.
