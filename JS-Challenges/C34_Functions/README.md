# JavaScript Challenge 34 — Higher-Order Functions and Arrow Functions

### **Task 1: Square Numbers**
Write a function `squareNumbers(arr)` using `map()` and arrow functions to return a new array where each number is squared.

### **Task 2: Filter Even Numbers**
Create a function `filterEvenNumbers(arr)` using `filter()` and arrow functions to return only the even numbers from an array.

### **Task 3: Sum of Positive Numbers**
Write a function `sumPositiveNumbers(arr)` that takes an array and returns the sum of all positive numbers using `filter()` and `reduce()` with arrow functions.

### **Task 4: Transform Array of Objects**
Write a function `getNames(arr)` that takes an array of objects (each having a `name` property) and returns an array containing only the names using `map()`.

### **Task 5: Find the Longest Word**
Write a function `findLongestWord(arr)` that takes an array of strings and returns the longest word using `reduce()`.
