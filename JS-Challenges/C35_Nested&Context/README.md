# JavaScript Challenge 35 — Nested Functions and Context

### **Task 1: Using `this` in Objects**
Create an object `person` with the following properties:
- `name`: A string value.
- `age`: A number value.
- `introduce()`: A method that returns the string: `"Hi, my name is [name] and I am [age] years old"`.

### **Task 2: Function within a Function**
Write a function `outer()` that contains a nested function `inner()`. When `outer()` is invoked, it should call `inner()` and return the string `"Inner function called"`.
